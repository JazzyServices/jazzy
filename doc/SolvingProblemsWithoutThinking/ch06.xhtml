Puzzle 6

On the edge of town, on a brownfield site, a new housing estate has been laid out consisting of sixteen plots of land. The Planning Committee have specified the following rules:
  1. Only one building per plot
  2. The building must be constructed in one of the four quadrants of the plot
    • top left
    • top right
    • bottom left
    • bottom right
  3. In the other three quadrants of a plot, there must be gardens.
  4. When a building has been constructed in a quadrant, all plots due north, south, east and west of this plot must have a garden in that particular quadrant (that is, on any row of plots (north-south or east-west) there mustn’t be two plots having buildings in the same quadrant).
  5. Furthermore, the corner plots must have buildings in their inner quadrants
  6. There must be a building on each plot.
The task of constructing the buildings has been given to the Gung-Ho Construction Company and the task of planting the gardens has been given to the Meticulous Landscaping Company. The CEO of the Meticulous Landscaping Company is concerned that the Gung-Ho Construction Company are going to construct buildings randomly and discover that it won’t be possible to have a building in each plot. “We will just knock some buildings down and build them in a different quadrant until we have got it right.” said Gung-Ho’s CEO.
The CEO of the Meticulous Landscaping Company thought this was a bad idea, so suggested, “For each building constructed, let us plant the gardens on the same plot, and in the same quadrant of the other plots in the same north-south row and in the same east-west row.” So the Landscaping Company did that (marked with circles on the map below).
The CEO of the Meticulous Landscaping Company then suggested, “Now construct a building in the top-left plot, because we know which quadrant it has to be in, and we will plant the corresponding gardens in the top-most and left-most plots.” After that was done, the map of the estate looked like this:

Puzzle 6 part 2

A farmer has a small orchard containing two apples trees (A), a cherry tree (C), a damson tree (D) and a beehive (B).
A D B C A
The farmer wants to plant more trees and get more beehives so that:
  1. The orchard has four rows and four columns
  2. Each row and each column contains exactly one each of an apple tree, a cherry tree, a damson tree and a beehive.
For each square in the orchard, the farmer makes a mark on the ground to indicate which item can’t go there, based on what items are already in the row and column of that square. That is, the farmer is eliminating the impossible.
  • If the square can’t contain an apple tree, make a mark in the top-left quadrant.
  • If the square can’t contain a beehive, make a mark in the top-right quadrant.
  • If the square can’t contain a cherry tree, make a mark in the bottom-left quadrant.
  • If the square can’t contain a damson tree, make a mark in the bottom-right quadrant.
The orchard now looks like this:
A D B C A
If a square contains three marks, then the farmer knows which item goes there because it is the only one possible. The farmer plants the right tree or places a beehive in that square then marks the other squares in the same row and column with a corresponding mark.
There are three squares on our map with three marks, so let’s put the correct letter in that square and mark the other squares in the same row and column.
A D A D C B C A
There are now two more squares with three marks. Write the correct letter in those squares, mark the corresponding squares in the same row and column, and repeat until you have filled the grid.
This sort of puzzle (where you must place one of each item in every row and column) is known as a Latin Square. The most common variant of this is the sudoku. But all Latin Square puzzles can be solved in the same way — by marking in the cell which items/numbers can’t go there. The marks need only be a small dot placed in a particular location within the cell. Marking the dots is like filling in the dead ends of a maze – it is all part of the meditative application of some simple rules that frees our brain from having to do any hard thinking.
