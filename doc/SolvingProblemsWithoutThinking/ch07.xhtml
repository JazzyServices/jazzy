Puzzle 7

This puzzle (sometimes called Number Blocks) isn’t a Latin Square puzzle, but we solve it in the same way (using marks to eliminate the impossible). The rules are defined as following:
  1. The grid is made of blocks (defined by the dark outlines).
  2. The blocks are made of between one and five cells.
  3. Each cell is to contain a number.
  4. A different number must appear in each cell in a block.
  5. The number in a cell must not be greater than the number of cells in its block
  6. Adjacent cells, even in different blocks and even diagonally must not contain the same number
3 2 5 1 1 4 5 3
In a similar way to the orchard puzzle we mark each cell in a particular way to indicate when a number can’t occupy that cell:
  • If the cell can’t contain a 1, make a mark in the top-left quadrant.
  • If the cell can’t contain a 2, make a mark in the top-right quadrant.
  • If the cell can’t contain a 3, make a mark in the bottom-left quadrant.
  • If the cell can’t contain a 4, make a mark in the bottom-right quadrant.
  • If the cell can’t contain a 5, make a mark in the middle.

Puzzle 7 step 1

First we apply rule 4 (a different number must appear in each cell in a block). So, if a block already contains the number 1, for example, we mark all the other cells in the block in the top-left.
3 2 5 1 1 4 5 3

Puzzle 7 step 2

We now apply rule 5 (the number in a cell must not be greater than the number of cells in its block). That is, a 4-cell block can not contain the number 5 (so mark all cells in this block in the middle), a 3-cell block can not contain the numbers 4 nor 5, so mark as appropriate; and so on. A 1-cell block can only contain the number 1, so we can write this number directly into the cell without having to mark it.
3 1 2 5 1 1 4 5 3

Puzzle 7 step 3

We now apply rule 6 (adjacent cells, even in different blocks and even diagonally must not contain the same number). That is, for each number already in the grid, mark its adjacent cells. Note, a cell may have up to eight neighbours.
3 1 2 5 1 1 4 5 3
We were hoping that some of the cells would contain four marks so that we could insert a number in that cell and move towards a solution. Does that mean we are stuck? Certainly not. Let’s look at rule 4 again: a different number must appear in each cell in a block. If all-but-one of the cells in a block contain the same mark, then the other cell must be the number that corresponds to that mark. For example, in the top-left block (containing a 3), two of the three un-numbered cells have a mark top-left indicating that they can’t be the number 1. Therefore, the cell without that mark must be a 1.

Puzzle 7 step 4

There are five such cells in the grid. Let’s enter the correct numbers into those cells and then follow rule 6 to mark adjacent cells.
3 1 2 1 1 5 1 2 1 4 4 1 5 3

Puzzle 7 step 5

Now we have a cell with four marks. So we write in the correct number and mark cells in the same block (rule 4) and all adjacent cells (rule 6). This will give us another 4-mark cell, so we complete that one also.
3 1 2 1 1 5 1 2 1 4 1 4 1 5 2 3

Puzzle 7 step 6

Now we have no more 4-mark cells but we do have blocks that have a single cell without a particular mark (as we had in step 4). Complete these cells, make the marks and repeat. If you have correctly eliminated the impossibles (using marks) and spotted the only-one-possibles, then you should be able to complete the grid.
