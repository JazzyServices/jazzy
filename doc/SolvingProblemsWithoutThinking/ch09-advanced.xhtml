Patterns used in Solving Latin Square Puzzles

There are two patterns that indicate where we can insert a number into a cell (because we have eliminated all the other possibilities)
  1. When a cell has N-1 marks.
  2. When the same mark appears in N-1 cells in the same row or column (or box in the case of sudoku).
Where N is the number of cells in a row or column (or box).
There are also several patterns that allow us to insert extra marks (because it is impossible for those cells to contain those numbers). We will show examples of each of these using sudoku, but they also apply (where appropriate) to futoshiki and other Latin Square puzzles.

Identical Twins

When two cells in the same set (row, column or box) have N-2 identical marks, then the other cells in the set can be marked with these two.
(Note, the grid below is a partial grid – we only show the numbers that illustrate our point. The other numbers have been omitted.)
2456 897 79 8
Notice that cells 2.1 and 3.1 both have the same seven marks. This means that these cells must be 1 and 3 (but we don't know yet which is which). Therefore the other cells in the row can’t be 1 or 3, so mark them accordingly.

Identical Triplets

When three cells in the same set have N-3 identical marks, then the other cells in the set can be marked with these three.
(Again, the grid below is a partial grid – we only show the numbers that illustrate our point. The other numbers have been omitted.)
68 4597 897 4 5
On the top row, cells 1.1, 1.2 and 2.3 all have the same six marks. This means that these cells must be 1, 2 and 3 (but we don't know yet which is which). Therefore the other cells in the row can’t be 1, 2 nor 3, so mark them accordingly.

Missing Twins

When the same two marks appear in N-2 cells in the same set; that is, if only two cells in the same set are missing the same two marks, mark the rest of these two cells.
(Again, the grid below is a partial grid – we only show the numbers that illustrate our point. The other numbers have been omitted.)
31 8 9 1 3
Cells 1.6 and 1.8 are the only cells in the box that don’t have a 1-mark and a 3-mark. This means that these cells must be 1 and 3 (but we don't know yet which is which). Therefore these cells can’t be 2, 4, 5, 6, 7, 8 nor 9, so mark them accordingly.

Missing Triplets

When the same three marks appear in N-3 cells in the same set; that is, if only three cells in the same set are missing the same three marks, mark the rest of these three cells.
(Again, the grid below is a partial grid – we only show the numbers that illustrate our point. The other numbers have been omitted.)
381 9 1 3 8
Cells 1.5, 1.6 and 1.8 are the only cells in the box that don’t have a 1-mark, a 3-mark and an 8-mark. This means that these cells must be 1, 3 and 8 (but we don't know yet which is which). Therefore these cells can’t be 2, 4, 5, 6, 7 nor 9, so mark them accordingly.

Non-identical Triplets and Quads

We have only included this section for completeness. Non-identical triplets are hard to spot so don’t really count as “without thinking”. Quads, whether identical or not are both rare and hard to spot.
The triplets we saw above still form a group even if one or more of them have N-2 marks rather than N-3.
68 4597 8973 4 5
Even though 2.3 has seven marks, cells 1.1, 1.2 and 2.3 all share the same six marks. This means that these cells must be 1, 2 and 3 (but we don't know yet which is which, though 2.3 can’t be 3). Therefore the other cells in the row can’t be 1, 2 nor 3, so we could mark them accordingly.

Sudoku Triads

A pattern that is parculiar to sudoku (and not to other Latin Square puzzles) is the triad. A triad is a set of three cells in the same box that are also in the same row or column.
If it is impossible for a number to be in two triads of a box it must, obviously, be in one of the cells in the other triad. Which means that it can’t be in the other two triads of its row/column.
45 87 6
In box 1, the only triad that the number 6 can appear is the 1,2,3-triad. Therefore, we can mark the other two triads of this row with the 6-mark.
45 87 6
Similarly, if it is impossible for a number to be in two triads of a row/column it must, obviously, be in one of the cells in the other triad. Which means that it can’t be in the other two triads of its box.

In the row running from 1.4 to 3.6, the only triad that the number 6 can appear is the one in box 3. Therefore, we can mark the other two triads of this box with the 6-mark.
